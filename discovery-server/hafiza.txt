## 1. SERVİSİN AMACI
Netflix Eureka Server tabanlı servis keşif merkezidir. Tüm mikroservisler
başlatıldığında kendilerini bu sunucuya kayıt eder (register). Servisler
arası iletişimde (FeignClient) hedef servisin adresini bulmak için
Eureka'ya sorgu yapılır. Eureka ayrıca Web Dashboard ile kayıtlı tüm
servislerin durumunu görsel olarak sunar.

Veritabanı YOKTUR. İş mantığı YOKTUR. Sadece altyapı servisidir.

## 2. KONFİGÜRASYON (application.yml)
```yaml
server:
  port: 8761

spring:
  application:
    name: discovery-server

eureka:
  instance:
    hostname: localhost
  client:
    register-with-eureka: false   # Kendini kendine kaydetme
    fetch-registry: false          # Başka Eureka sunucusundan kayıt çekme

management:
  endpoints:
    web:
      exposure:
        include: health,prometheus
  endpoint:
    health:
      show-details: always
  prometheus:
    metrics:
      export:
        enabled: true
```

Kritik ayarlar:
- register-with-eureka: false → Kendini kendi registry'sine kaydetmez
- fetch-registry: false → Dış bir Eureka kümesinden veri çekmez (Tek sunucu modu)
